gamvas.Sound

Description

A class for sound and music files.

Constructur

new gamvas.Actor(file);

Parameters

filea JavaScript Audio class

See

gamvas.State.addSound

Example

myState = gamvas.State.extend({
    init: function() {
        this.sound = this.addSound("introsound.wav");
    },
    onKeyDown: function() {
        this.sound.play();
    }
});
