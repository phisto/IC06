Cliquez sur les boutons pour choisir un élément puis cliquez sur le jeu pour le placer ou vous le désirez. Vous pouvez ensuite le redéplacer en cliquant dessus (bien au milieu). L'objet sera alors selectionné. Pour déselectionner, recliquez dessus. Si l'objet est selectionné, vous pouvez le supprimer avec la touche Backspace/Retour. Lorsqu'un mur est selectionné, vous pouvez le tourner avec els touches gauche et droite. Vous pouvez sauver votre niveau en cliquant sur le bouton "Save" puis en donnant un numéro de niveau.
